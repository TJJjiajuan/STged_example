normalization of data

note: In the predicted cell type-specific gene expression data, we applied a normalization procedure to ensure that, for each spot, the total gene expression level per cell type sums to 1. Specifically, for a given cell type \(K\), the predicted cell type-specific gene expression is represented as a tensor \(F\), with dimensions \(p \times n \times K\), where \(p\) is the number of genes, \(n\) the number of spatial spots, and \(K\) the number of cell types. To normalize the expression data, for each cell type \(k\), we calculated the sum of gene expression at each spot \(j\), and divided the expression of each gene \(i\) by this sum: \[ F^{(k)}_{i,j} = \frac{F_{i,j,k}}{\sum_{i=1}^{p} F_{i,j,k}} \] This normalization ensures that the total expression level of all genes at each spot is equal to 1, maintaining the relative expression levels of genes while standardizing the total expression per spot. The normalization was performed independently for each cell type, ensuring consistency in the interpretation of spatial gene expression patterns.
